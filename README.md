# RAPLSTAT
A simple tool to calculate power draw, e.g. for use in conky.

## De-/Installation
Run `make` to compile, `make install` to install, `make uninstall` to uninstall.

## Usage
Run `raplstat` to calculate power draw. You can specify a custom path to the respective "energy_uj" if you want. Output is in watts.

Please note you may need root rights to run this. See [https://www.heise.de/news/Platypus-Sicherheitsluecke-missbraucht-Messfunktion-von-Intel-Prozessoren-4953484.html]
